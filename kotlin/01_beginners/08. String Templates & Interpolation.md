# String Templates & Interpolation

## Overview

Kotlin provides powerful string templating capabilities that make string formatting concise and readable. This guide covers string interpolation, raw strings, and string manipulation.

## Deep Explanation

### String Interpolation

#### Basic Interpolation

```kotlin
val name = "Kotlin"
val message = "Hello, $name!"  // "Hello, Kotlin!"
```

#### Expression Interpolation

```kotlin
val a = 10
val b = 20
val sum = "Sum: ${a + b}"  // "Sum: 30"
```

#### Complex Expressions

```kotlin
val user = User("Alice", 25)
val info = "Name: ${user.name}, Age: ${user.age}, Next Year: ${user.age + 1}"
```

#### Property Access

```kotlin
val length = "Kotlin".length
val message = "Length is $length"  // "Length is 6"
```

### Raw Strings (Triple Quotes)

#### Multiline Strings

```kotlin
val text = """
    This is a
    multiline string
    with multiple lines
""".trimIndent()
```

#### Preserve Formatting

```kotlin
val sql = """
    SELECT * FROM users
    WHERE age > 18
    ORDER BY name
""".trimIndent()
```

#### Without trimIndent

```kotlin
val text = """
    |Line 1
    |Line 2
    |Line 3
""".trimMargin()
```

### String Functions

#### Common Operations

```kotlin
val text = "Hello Kotlin"

text.uppercase()              // "HELLO KOTLIN"
text.lowercase()              // "hello kotlin"
text.capitalize()             // "Hello kotlin"
text.decapitalize()           // "hello Kotlin"

text.length                    // 12
text.isEmpty()                // false
text.isBlank()                // false (checks whitespace)
```

#### Substring & Splitting

```kotlin
val text = "Hello, World, Kotlin"

text.substring(0, 5)          // "Hello"
text.split(", ")               // [Hello, World, Kotlin]
text.split(",").map { it.trim() }
```

#### Replacement

```kotlin
val text = "Hello World"

text.replace("World", "Kotlin")        // "Hello Kotlin"
text.replaceFirst("l", "L")            // "HeLlo World"
text.replace("l", "L", ignoreCase = true)
```

#### Checking

```kotlin
val text = "Hello123"

text.startsWith("Hello")       // true
text.endsWith("123")           // true
text.contains("lo")            // true
text.matches(Regex("\\d+"))    // false
```

### String Builders

```kotlin
val builder = StringBuilder()
builder.append("Hello")
builder.append(" ")
builder.append("Kotlin")
val result = builder.toString()  // "Hello Kotlin"

// Or use buildString
val result = buildString {
    append("Hello")
    append(" ")
    append("Kotlin")
}
```

### Examples

#### Example 1: Formatted Output

```kotlin
fun formatUser(user: User): String {
    return """
        User Information:
        - Name: ${user.name}
        - Age: ${user.age}
        - Email: ${user.email ?: "Not provided"}
    """.trimIndent()
}
```

#### Example 2: Template Function

```kotlin
fun greet(name: String, age: Int): String {
    return "Hello, $name! You are $age years old. Next year you'll be ${age + 1}."
}
```

## Summary

Kotlin string templates are powerful and concise:
- `$variable` for simple interpolation
- `${expression}` for complex expressions
- Triple quotes for multiline strings
- Rich string manipulation functions
