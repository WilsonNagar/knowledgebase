# Status bar service

## Overview

Status bar service refers to the service component within SystemUI that manages the status bar, including status bar visibility, status bar content updates, system icon management, notification icon display, clock updates, and status bar interactions. The status bar service coordinates with system services to display system information and provides the status bar UI that is visible across all Android applications. Understanding status bar service is essential for AOSP development, as it explains how the status bar is managed, how status bar content is updated, how system icons are displayed, how notifications are shown, and how status bar service integrates with system services. This guide provides a comprehensive overview of status bar service, service architecture, content management, icon management, system integration, and service lifecycle.

Think of status bar service like a control center for the status bar: just as a control center manages and coordinates various displays (status bar service), monitors system information (system icons, notifications), and updates displays in real-time (status bar updates), the status bar service manages and coordinates the status bar display, monitors system information, and updates the status bar in real-time.
