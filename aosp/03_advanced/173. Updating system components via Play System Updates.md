# Updating system components via Play System Updates

## Overview

Updating system components via Play System Updates refers to the mechanism by which Android system components (APEX modules and system APKs) are updated through Google Play System Updates, enabling independent component updates without requiring full system updates. Understanding how to update system components via Play System Updates is essential for AOSP development, as it explains how Play System Updates work, how updates are delivered, how updates are installed, how updates are verified, and how this mechanism enables faster security and feature updates. This guide provides a comprehensive overview of Play System Updates, update delivery, update installation, update verification, update management, and update benefits.

Think of Play System Updates like app store updates for system components: just as you can update individual apps through an app store without updating the entire operating system, Play System Updates allow you to update individual system components (like security patches, media codecs, or network components) through Google Play without requiring a full Android system update. This enables faster delivery of critical updates and improvements.

## Deep Explanation

### What are Play System Updates?

Play System Updates refer to the mechanism by which Android system components are updated through Google Play, enabling independent component updates without requiring full system updates. Play System Updates deliver APEX modules and system APKs.

**Key Characteristics:**

- **Play Delivery:** Delivered via Google Play
- **Independent:** Independent of Android version
