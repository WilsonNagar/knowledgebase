## Overview

Embedded systems operating systems are designed for resource-constrained devices with specific requirements. Understanding embedded OS architectures, RTOS for embedded systems, resource management, and embedded system design is essential for building efficient embedded applications.

## Quiz

1. What is an embedded system?
   - **A)** Desktop computer
   - **B)** Computer system embedded in larger system with resource constraints
   - **C)** Server
   - **D)** Supercomputer

2. What is FreeRTOS?
   - **A)** Desktop OS
   - **B)** Popular open-source RTOS for embedded systems
   - **C)** Server OS
   - **D)** Database

3. What are key constraints in embedded systems?
   - **A)** Unlimited resources
   - **B)** Limited memory, CPU, power, and size
   - **C)** No constraints
   - **D)** Only memory

**Answers:**
1. **B** - Embedded systems are dedicated computer systems within larger systems, typically with limited resources and specific functions
2. **B** - FreeRTOS is a popular open-source real-time operating system designed for embedded systems with small footprint
3. **B** - Embedded systems face constraints in memory (RAM/ROM), CPU power, battery/power consumption, and physical size
