## Set Operations Advanced

### Advanced Operations

**1. Power Set**:
```
P(S) = {all subsets of S}
```

**2. Cartesian Product**:
```
A × B = {(a,b) | a ∈ A, b ∈ B}
```

**3. Set Difference**:
```
A - B = {x | x ∈ A and x ∉ B}
```

**4. Symmetric Difference**:
```
A Δ B = (A - B) ∪ (B - A)
```

### Power Set Example

**Example**: 
```
S = {1, 2}
P(S) = {∅, {1}, {2}, {1,2}}
```

**Size**: |P(S)| = 2^|S|

**Use**: All possible subsets

## Relations

### What are Relations?

**Relation**: Set of ordered pairs

**Definition**: 
```
R ⊆ A × B
```

**Types**: 
- **Reflexive**: (a,a) ∈ R
- **Symmetric**: (a,b) ∈ R → (b,a) ∈ R
- **Transitive**: (a,b), (b,c) ∈ R → (a,c) ∈ R
- **Equivalence**: Reflexive, symmetric, transitive

**Use**: Database relations, equivalence classes

### Equivalence Relations

**Equivalence Relation**: 
- **Reflexive**: a ~ a
- **Symmetric**: a ~ b → b ~ a
- **Transitive**: a ~ b, b ~ c → a ~ c

**Equivalence Classes**: 
```
[a] = {b | a ~ b}
```

**Use**: Partition sets

## Functions

### What are Functions?

**Function**: Special relation

**Definition**: 
```
f: A → B
For each a ∈ A, exactly one b ∈ B
```

**Types**: 
- **Injective**: One-to-one
- **Surjective**: Onto
- **Bijective**: One-to-one and onto

**Use**: Mappings, transformations

### Function Types

**Injective (One-to-One)**:
```
f(a1) = f(a2) → a1 = a2
```

**Surjective (Onto)**:
```
For each b ∈ B, exists a ∈ A: f(a) = b
```

**Bijective**:
```
Both injective and surjective
```

**Use**: Isomorphisms, bijections

## Cardinality

### What is Cardinality?

**Cardinality**: Size of set

**Finite Sets**: 
```
|S| = number of elements
```

**Infinite Sets**: 
```
Countable vs uncountable
```

**Comparison**: 
```
|A| = |B| if bijection exists
```

**Use**: Compare set sizes

### Countable Sets

**Countable**: 
- **Finite**: Finite sets
- **Countably Infinite**: Bijection with ℕ

**Examples**: 
- **ℕ**: Natural numbers
- **ℤ**: Integers
- **ℚ**: Rational numbers

**Uncountable**: 
- **ℝ**: Real numbers
- **P(ℕ)**: Power set of naturals

## Infinite Sets

### Infinite Set Properties

**Infinite**: Not finite

**Types**: 
- **Countably Infinite**: Same size as ℕ
- **Uncountably Infinite**: Larger than ℕ

**Cantor's Theorem**: 
```
|S| < |P(S)|
```

**Implication**: Infinitely many infinities

### Cantor's Diagonal Argument

**Diagonal Argument**: 
```
Prove ℝ is uncountable
```

**Method**: 
```
Assume countable list
Construct number not in list
Contradiction
```

**Result**: ℝ uncountable

## Set Theory Axioms

### ZFC Axioms

**ZFC**: Zermelo-Fraenkel with Choice

**Axioms**: 
- **Extensionality**: Sets equal if same elements
- **Pairing**: {a,b} exists
- **Union**: Union exists
- **Power Set**: Power set exists
- **Infinity**: Infinite set exists
- **Choice**: Axiom of Choice

**Foundation**: Set theory foundation

## Quiz

1. What is a power set?
   - **A)** Subset
   - **B)** Set of all subsets of a set
   - **C)** Empty set
   - **D)** Single element

2. What is an equivalence relation?
   - **A)** Any relation
   - **B)** Relation that is reflexive, symmetric, and transitive
   - **C)** Function
   - **D)** Set

3. What is cardinality?
   - **A)** Order
   - **B)** Size of set (number of elements or bijection type)
   - **C)** Relation
   - **D)** Function

**Answers:**
1. **B** - Power set P(S) contains all possible subsets of set S, with size 2^|S|
2. **B** - Equivalence relations partition sets into equivalence classes through reflexive, symmetric, and transitive properties
3. **B** - Cardinality measures set size: finite sets use element count, infinite sets use bijections (countable vs uncountable)
