## What is Combinatorics?

### Definition

**Combinatorics**: Study of counting and arrangements

**Topics**: 
- **Counting**: Count arrangements
- **Permutations**: Ordered arrangements
- **Combinations**: Unordered selections
- **Structures**: Combinatorial structures

**Use**: Algorithm analysis, probability, optimization

### Combinatorial Problems

**Types**: 
- **Counting**: How many ways?
- **Existence**: Does arrangement exist?
- **Optimization**: Best arrangement?
- **Enumeration**: List all arrangements

## Permutations

### What are Permutations?

**Permutation**: Ordered arrangement

**Formula**: 
```
P(n,r) = n! / (n-r)!
```

**Example**: 
```
Permutations of {1,2,3}:
123, 132, 213, 231, 312, 321
```

**Use**: Ordering problems

### Permutations with Repetition

**With Repetition**: 
```
n^r arrangements
```

**Example**: 
```
3-digit numbers: 10^3 = 1000
```

**Use**: Strings, sequences

## Combinations

### What are Combinations?

**Combination**: Unordered selection

**Formula**: 
```
C(n,r) = n! / (r!(n-r)!)
```

**Notation**: 
```
C(n,r) = (n choose r) = nCr
```

**Example**: 
```
Combinations of {1,2,3} choose 2:
{1,2}, {1,3}, {2,3}
```

**Use**: Selection problems

### Combinations Properties

**Symmetry**: 
```
C(n,r) = C(n, n-r)
```

**Pascal's Identity**: 
```
C(n,r) = C(n-1,r-1) + C(n-1,r)
```

**Use**: Combinatorial identities

## Binomial Coefficients

### What are Binomial Coefficients?

**Binomial Coefficient**: C(n,r)

**Binomial Theorem**: 
```
(x+y)^n = Σ C(n,k) x^k y^(n-k)
```

**Use**: Expand binomials, counting

### Pascal's Triangle

**Pascal's Triangle**: 
```
    1
   1 1
  1 2 1
 1 3 3 1
1 4 6 4 1
```

**Property**: 
```
Each entry = sum of two above
```

**Use**: Compute binomial coefficients

## Inclusion-Exclusion

### What is Inclusion-Exclusion?

**Inclusion-Exclusion**: Count union of sets

**Formula**: 
```
|A ∪ B| = |A| + |B| - |A ∩ B|
```

**General**: 
```
|A₁ ∪ ... ∪ Aₙ| = 
  Σ|Aᵢ| - Σ|Aᵢ ∩ Aⱼ| + 
  Σ|Aᵢ ∩ Aⱼ ∩ Aₖ| - ...
```

**Use**: Count with overlaps

### Inclusion-Exclusion Example

**Example**: Count numbers divisible by 2 or 3
```
|A ∪ B| = |A| + |B| - |A ∩ B|
       = 50 + 33 - 16
       = 67
```

**Use**: Solve counting problems

## Generating Functions

### What are Generating Functions?

**Generating Function**: Power series

**Definition**: 
```
G(x) = Σ aₙ xⁿ
```

**Use**: 
- **Counting**: Encode sequences
- **Recurrences**: Solve recurrences
- **Combinatorics**: Combinatorial problems

### Generating Function Example

**Example**: 
```
(1+x)^n = Σ C(n,k) x^k
```

**Use**: Encode combinations

## Pigeonhole Principle

### What is Pigeonhole Principle?

**Pigeonhole Principle**: 
```
If n+1 objects in n boxes,
at least one box has 2+ objects
```

**Use**: Existence proofs

### Pigeonhole Example

**Example**: 
```
13 people → 12 months
At least 2 share birthday month
```

**Use**: Prove existence

## Quiz

1. What is a permutation?
   - **A)** Unordered selection
   - **B)** Ordered arrangement of objects
   - **C)** Set
   - **D)** Function

2. What is a combination?
   - **A)** Ordered arrangement
   - **B)** Unordered selection of objects
   - **C)** Permutation
   - **D)** Sequence

3. What is the Pigeonhole Principle?
   - **A)** Counting formula
   - **B)** If n+1 objects in n boxes, at least one box has 2+ objects
   - **C)** Probability theorem
   - **D)** Set theory

**Answers:**
1. **B** - Permutations are ordered arrangements where order matters (P(n,r) = n!/(n-r)!)
2. **B** - Combinations are unordered selections where order doesn't matter (C(n,r) = n!/(r!(n-r)!))
3. **B** - Pigeonhole Principle guarantees that if you place more objects than containers, at least one container must hold multiple objects
