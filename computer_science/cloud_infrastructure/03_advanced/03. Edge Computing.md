# Edge Computing

## Overview

Edge computing brings computation and data storage closer to data sources, reducing latency and bandwidth usage. Understanding edge computing architecture, edge devices, edge platforms, and use cases is essential for low-latency applications, IoT, and real-time processing.

## What is Edge Computing?

### Definition

**Edge Computing**: Compute at edge

**Purpose**: Process data closer to source

**Location**: Edge of network

**Benefit**: Lower latency, reduced bandwidth

### Edge Computing Benefits

**1. Low Latency**:
```
Process closer to source
```

**2. Reduced Bandwidth**:
```
Process locally, send less data
```

**3. Privacy**:
```
Process sensitive data locally
```

**4. Offline Capability**:
```
Work offline
```

**5. Real-Time Processing**:
```
Real-time processing
```

## Edge vs Cloud Computing

### Comparison

| Aspect | Cloud Computing | Edge Computing |
|--------|----------------|----------------|
| **Location** | Centralized data centers | Edge locations |
| **Latency** | Higher latency | Lower latency |
| **Bandwidth** | High bandwidth usage | Reduced bandwidth |
| **Processing** | Centralized | Distributed |
| **Use Cases** | General computing | Low-latency, IoT |

### When to Use Edge

**Use Edge When**:
- **Low Latency**: Low latency required
- **Bandwidth**: Limited bandwidth
- **Privacy**: Privacy concerns
- **Offline**: Offline capability needed
- **Real-Time**: Real-time processing

## Quiz

1. What is edge computing?
   - **A)** Cloud computing
   - **B)** Computing at edge locations closer to data sources
   - **C)** Centralized computing
   - **D)** Distributed computing

2. What is the main benefit of edge computing?
   - **A)** Lower cost
   - **B)** Lower latency and reduced bandwidth usage
   - **C)** Simpler architecture
   - **D)** More resources

3. What enables edge computing?
   - **A)** 4G
   - **B)** 5G networks enable edge computing with low latency
   - **C)** WiFi
   - **D)** Ethernet

**Answers:**
1. **B** - Edge computing processes data at edge locations (closer to data sources) rather than in centralized cloud data centers, reducing latency and bandwidth
2. **B** - Main benefits are lower latency (processing closer to source) and reduced bandwidth usage (processing locally, sending less data to cloud)
3. **B** - 5G networks with Mobile Edge Computing (MEC) enable edge computing by providing ultra-low latency and high bandwidth at edge locations
