## What is Computer Vision?

### Definition

**Computer Vision**: Understand visual information

**Goal**: 
- **Recognize**: Recognize objects
- **Detect**: Detect objects
- **Segment**: Segment images
- **Understand**: Understand scenes

**Challenges**: 
- **Variation**: Viewpoint, lighting, scale
- **Occlusion**: Objects occluded
- **Background**: Complex backgrounds

### Computer Vision Tasks

**1. Classification**: 
```
Classify images
```

**2. Object Detection**: 
```
Detect and localize objects
```

**3. Segmentation**: 
```
Segment images
```

**4. Recognition**: 
```
Recognize objects
```

**5. Tracking**: 
```
Track objects
```

## Convolutional Neural Networks

### What are CNNs?

**CNN**: Convolutional Neural Network

**Components**: 
- **Convolutional Layers**: Feature extraction
- **Pooling Layers**: Downsampling
- **Fully Connected**: Classification

**Properties**: 
- **Translation Invariant**: Invariant to translation
- **Parameter Sharing**: Shared weights
- **Spatial Hierarchy**: Hierarchical features

**Use**: Image classification

### Convolution Operation

**Convolution**: 
```
Apply filter (kernel) to image
Slide filter across image
Compute dot product
```

**Output**: 
```
Feature map
```

**Use**: Extract features

### Pooling

**Pooling**: 
- **Max Pooling**: Take maximum
- **Average Pooling**: Take average

**Purpose**: 
- **Downsampling**: Reduce size
- **Translation Invariant**: Invariance

**Use**: Reduce dimensions

## CNN Architectures

### LeNet

**LeNet**: Early CNN

**Architecture**: 
```
Conv → Pool → Conv → Pool → FC → FC
```

**Use**: Digit recognition

### AlexNet

**AlexNet**: Deep CNN breakthrough

**Architecture**: 
```
5 Conv layers + 3 FC layers
ReLU activation
Dropout
```

**Use**: ImageNet classification

### VGG

**VGG**: Very deep network

**Architecture**: 
```
Small 3×3 filters
Deep layers (16-19)
```

**Use**: Image classification

### ResNet

**ResNet**: Residual networks

**Innovation**: 
```
Skip connections
Residual blocks
```

**Benefit**: 
```
Train very deep networks
```

**Use**: State-of-the-art classification

## Object Detection

### What is Object Detection?

**Object Detection**: 
```
Detect and localize objects
```

**Output**: 
```
Bounding boxes + classes
```

**Methods**: 
- **Two-Stage**: R-CNN, Fast R-CNN, Faster R-CNN
- **One-Stage**: YOLO, SSD

**Use**: Detect multiple objects

### YOLO

**YOLO**: You Only Look Once

**Method**: 
```
Single pass through network
Predict boxes and classes
```

**Benefit**: 
```
Fast detection
```

**Use**: Real-time detection

### R-CNN Family

**R-CNN**: 
```
Region proposals → CNN → Classification
```

**Fast R-CNN**: 
```
Shared computation
```

**Faster R-CNN**: 
```
End-to-end training
```

**Use**: Accurate detection

## Transfer Learning

### What is Transfer Learning?

**Transfer Learning**: 
```
Use pre-trained models
Fine-tune for new task
```

**Benefit**: 
```
Less data needed
Faster training
```

**Use**: Limited data scenarios

### Transfer Learning Process

**Process**: 
```
1. Use pre-trained model (ImageNet)
2. Remove last layer
3. Add new classification layer
4. Fine-tune on new data
```

**Use**: Custom tasks

## Quiz

1. What is a CNN?
   - **A)** Recurrent network
   - **B)** Convolutional Neural Network for image processing
   - **C)** Transformer
   - **D)** MLP

2. What is object detection?
   - **A)** Classification only
   - **B)** Detect and localize objects with bounding boxes
   - **C)** Segmentation only
   - **D)** Tracking only

3. What is transfer learning?
   - **A)** Train from scratch
   - **B)** Use pre-trained models and fine-tune for new tasks
   - **C)** No learning
   - **D)** Random initialization

**Answers:**
1. **B** - CNNs use convolutional layers for feature extraction, pooling for downsampling, and are translation-invariant
2. **B** - Object detection identifies objects and their locations using bounding boxes, unlike classification which only labels images
3. **B** - Transfer learning leverages pre-trained models (often on ImageNet) and fine-tunes them for new tasks with limited data
