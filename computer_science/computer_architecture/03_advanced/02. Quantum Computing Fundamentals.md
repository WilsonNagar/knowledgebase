## What is Quantum Computing?

### Definition

**Quantum Computing**: Computing using quantum mechanics

**Principles**: 
- **Superposition**: Qubits in superposition
- **Entanglement**: Entangled qubits
- **Interference**: Quantum interference
- **Measurement**: Measurement collapses state

**Benefit**: Potential exponential speedup for certain problems

**Challenge**: Error correction, decoherence

## Quantum Bits (Qubits)

### What is a Qubit?

**Qubit**: Quantum bit

**Classical Bit**: 0 or 1

**Qubit**: Superposition of 0 and 1

**State**: |ψ⟩ = α|0⟩ + β|1⟩

**Where**: |α|² + |β|² = 1

**Measurement**: Collapses to 0 or 1

### Qubit States

**Basis States**:
- **|0⟩**: State 0
- **|1⟩**: State 1

**Superposition**:
- **|+⟩**: (|0⟩ + |1⟩)/√2
- **|-⟩**: (|0⟩ - |1⟩)/√2

**Bloch Sphere**: Visualize qubit state

## Quantum Superposition

### What is Superposition?

**Superposition**: Qubit in multiple states

**Classical**: Bit is 0 OR 1

**Quantum**: Qubit is 0 AND 1 (superposition)

**Measurement**: Collapses to 0 or 1

### Superposition Example

**Single Qubit**:
```
|ψ⟩ = (|0⟩ + |1⟩)/√2
```

**Measurement**: 
- **Probability 0**: 50%
- **Probability 1**: 50%

**Multiple Qubits**:
```
|ψ⟩ = (|00⟩ + |01⟩ + |10⟩ + |11⟩)/2
```

**Measurement**: Equal probability of all states

## Quantum Entanglement

### What is Entanglement?

**Entanglement**: Qubits correlated

**Property**: Measurement of one affects other

**Example**: 
```
|ψ⟩ = (|00⟩ + |11⟩)/√2
```

**Measurement**: 
- **If first is 0**: Second is 0
- **If first is 1**: Second is 1

**Use**: Quantum algorithms, teleportation

### Bell States

**Bell States**: Maximally entangled states

**Examples**:
- **|Φ⁺⟩**: (|00⟩ + |11⟩)/√2
- **|Φ⁻⟩**: (|00⟩ - |11⟩)/√2
- **|Ψ⁺⟩**: (|01⟩ + |10⟩)/√2
- **|Ψ⁻⟩**: (|01⟩ - |10⟩)/√2

### Common Quantum Gates

**Hadamard (H)**:
```
H|0⟩ = (|0⟩ + |1⟩)/√2
H|1⟩ = (|0⟩ - |1⟩)/√2
```

**Pauli-X (NOT)**:
```
X|0⟩ = |1⟩
X|1⟩ = |0⟩
```

**CNOT**:
```
CNOT|00⟩ = |00⟩
CNOT|01⟩ = |01⟩
CNOT|10⟩ = |11⟩
CNOT|11⟩ = |10⟩
```

## Quantum Algorithms

### Shor's Algorithm

**Purpose**: Factor integers

**Speedup**: Exponential

**Impact**: Breaks RSA encryption

**Complexity**: O((log N)³)

### Grover's Algorithm

**Purpose**: Search unsorted database

**Speedup**: Quadratic

**Complexity**: O(√N)

**Use**: Database search, optimization

### Quantum Fourier Transform

**Purpose**: Quantum version of FFT

**Use**: Shor's algorithm, phase estimation

**Speedup**: Exponential

## Quantum vs Classical Computing

### Comparison

| Aspect | Classical | Quantum |
|--------|-----------|---------|
| **Bits** | 0 or 1 | Superposition |
| **Operations** | Deterministic | Probabilistic |
| **Measurement** | Non-destructive | Destructive |
| **Entanglement** | No | Yes |
| **Speedup** | N/A | Exponential (some problems) |

### When Quantum Wins

**Quantum Advantages**:
- **Factoring**: Shor's algorithm
- **Search**: Grover's algorithm
- **Simulation**: Quantum simulation
- **Optimization**: Quantum optimization

**Classical Advantages**:
- **General Purpose**: Most problems
- **Error Rate**: Lower error rate
- **Maturity**: Mature technology
- **Cost**: Lower cost

## Quiz

1. What is a qubit?
   - **A)** Classical bit
   - **B)** Quantum bit that can be in superposition of 0 and 1
   - **C)** Quantum gate
   - **D)** Quantum algorithm

2. What is quantum superposition?
   - **A)** Single state
   - **B)** Qubit existing in multiple states simultaneously until measured
   - **C)** Entanglement
   - **D)** Measurement

3. What is quantum entanglement?
   - **A)** Independent qubits
   - **B)** Qubits correlated such that measurement of one affects the other
   - **C)** Superposition
   - **D)** Quantum gate

**Answers:**
1. **B** - A qubit (quantum bit) can exist in a superposition of |0⟩ and |1⟩ states, represented as α|0⟩ + β|1⟩ where |α|² + |β|² = 1
2. **B** - Quantum superposition allows a qubit to exist in multiple states simultaneously (0 and 1) until measurement collapses it to a definite state
3. **B** - Quantum entanglement creates correlations between qubits such that measuring one qubit instantly affects the state of the other, even at distance
