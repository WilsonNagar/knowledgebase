## Early Computing

### Mechanical Calculators

**Abacus** (c. 2400 BC):
```
First calculating device
Beads on rods
```

**Pascaline** (1642):
```
Blaise Pascal's mechanical calculator
Addition and subtraction
```

**Difference Engine** (1822):
```
Charles Babbage's design
Polynomial calculations
```

**Analytical Engine** (1837):
```
Babbage's programmable computer design
Never fully built
```

### Early Concepts

**Ada Lovelace** (1843):
```
First programmer
Wrote algorithm for Analytical Engine
```

**Boolean Algebra** (1854):
```
George Boole's logic system
Foundation for digital circuits
```

## The First Computers

### Electronic Computers

**ENIAC** (1946):
```
First general-purpose electronic computer
18,000 vacuum tubes
```

**UNIVAC** (1951):
```
First commercial computer
Used for census
```

**IBM 701** (1952):
```
IBM's first scientific computer
```

### Stored Program Concept

**EDVAC** (1949):
```
Stored program concept
Von Neumann architecture
```

**Key Innovation**: 
```
Programs stored in memory
Same memory for data and instructions
```

## Quiz

1. Who is considered the first programmer?
   - **A)** Charles Babbage
   - **B)** Ada Lovelace
   - **C)** Alan Turing
   - **D)** Grace Hopper

2. What was the first general-purpose electronic computer?
   - **A)** UNIVAC
   - **B)** ENIAC
   - **C)** IBM 701
   - **D)** EDVAC

3. Who invented the World Wide Web?
   - **A)** Bill Gates
   - **B)** Tim Berners-Lee
   - **C)** Larry Page
   - **D)** Steve Jobs

**Answers:**
1. **B** - Ada Lovelace wrote the first algorithm for Babbage's Analytical Engine in 1843
2. **B** - ENIAC (Electronic Numerical Integrator and Computer) was completed in 1946
3. **B** - Tim Berners-Lee invented the World Wide Web in 1991 at CERN
