## Prime Numbers

### What are Primes?

**Prime**: Number > 1 divisible only by 1 and itself

**Examples**: 2, 3, 5, 7, 11, 13, 17, 19, 23

**Composite**: Not prime (e.g., 4, 6, 8, 9, 10)

### Prime Properties

**1. Fundamental Theorem of Arithmetic**:
```
Every integer > 1 can be uniquely factored into primes
```

**2. Infinitely Many Primes**:
```
There are infinitely many primes
```

**3. Prime Testing**:
```
Check divisibility up to √n
```

### Prime Testing

**Naive**:
```c
bool is_prime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
```

**Time**: O(√n)

## Divisibility

### Divisibility Rules

**Definition**: a divides b (a | b) if b = a × k for some integer k

**Properties**:
- **Transitive**: If a | b and b | c, then a | c
- **Linear Combination**: If a | b and a | c, then a | (mb + nc)

**Examples**:
```
3 | 12 (12 = 3 × 4)
5 | 15 (15 = 5 × 3)
```

## GCD & LCM

### Greatest Common Divisor (GCD)

**Definition**: Largest number dividing both

**Notation**: gcd(a, b)

**Examples**:
```
gcd(12, 18) = 6
gcd(7, 13) = 1 (coprime)
```

### Least Common Multiple (LCM)

**Definition**: Smallest number divisible by both

**Notation**: lcm(a, b)

**Formula**: lcm(a, b) = (a × b) / gcd(a, b)

**Examples**:
```
lcm(4, 6) = 12
lcm(3, 5) = 15
```

## Modular Arithmetic

### What is Modular Arithmetic?

**Modulo**: Remainder after division

**Notation**: a mod n = remainder when a divided by n

**Examples**:
```
7 mod 3 = 1
10 mod 4 = 2
15 mod 5 = 0
```

### Modular Properties

**1. Addition**:
```
(a + b) mod n = ((a mod n) + (b mod n)) mod n
```

**2. Multiplication**:
```
(a × b) mod n = ((a mod n) × (b mod n)) mod n
```

**3. Exponentiation**:
```
a^b mod n = (a mod n)^b mod n
```

### Modular Inverse

**Definition**: a^(-1) mod n such that a × a^(-1) ≡ 1 (mod n)

**Exists**: If gcd(a, n) = 1

**Example**:
```
3^(-1) mod 7 = 5 (since 3 × 5 = 15 ≡ 1 mod 7)
```

## Euclidean Algorithm

### GCD Algorithm

**Euclidean Algorithm**: Compute GCD efficiently

**Process**:
```
gcd(a, b) = gcd(b, a mod b)
Repeat until b = 0
Then gcd = a
```

**Example**:
```
gcd(48, 18):
  gcd(48, 18) = gcd(18, 48 mod 18) = gcd(18, 12)
  gcd(18, 12) = gcd(12, 18 mod 12) = gcd(12, 6)
  gcd(12, 6) = gcd(6, 12 mod 6) = gcd(6, 0)
  gcd(6, 0) = 6
```

### Extended Euclidean Algorithm

**Purpose**: Find x, y such that ax + by = gcd(a, b)

**Use**: Compute modular inverse

**Algorithm**:
```
ExtendedGCD(a, b):
  if b == 0: return (a, 1, 0)
  (g, x1, y1) = ExtendedGCD(b, a mod b)
  x = y1
  y = x1 - (a // b) * y1
  return (g, x, y)
```

## Fermat's Little Theorem

### Theorem

**Statement**: If p is prime and gcd(a, p) = 1, then:
```
a^(p-1) ≡ 1 (mod p)
```

**Corollary**: 
```
a^(-1) mod p = a^(p-2) mod p
```

**Use**: Compute modular inverse

### Example

**Compute 3^(-1) mod 7**:
```
Since 7 is prime:
3^(-1) mod 7 = 3^(7-2) mod 7 = 3^5 mod 7
3^5 = 243 ≡ 5 mod 7
So 3^(-1) mod 7 = 5
```

## Chinese Remainder Theorem

### Theorem

**Statement**: Given pairwise coprime moduli n₁, n₂, ..., nₖ and remainders a₁, a₂, ..., aₖ, there exists unique solution modulo N = n₁ × n₂ × ... × nₖ

**Example**:
```
x ≡ 2 (mod 3)
x ≡ 3 (mod 5)
x ≡ 2 (mod 7)

Solution: x ≡ 23 (mod 105)
```

**Use**: Solve systems of congruences

## Real-World Examples

### Example 1: RSA Encryption

**Process**:
```
1. Choose primes p, q
2. n = p × q
3. Public key: (n, e)
4. Private key: (n, d) where e × d ≡ 1 mod φ(n)
```

**Security**: Based on difficulty of factoring

### Example 2: Hash Tables

**Hash Function**: 
```
h(k) = k mod m
```

**Collision**: When k₁ mod m = k₂ mod m

## Common Pitfalls

### Problem: Integer Overflow

```c
// BAD: May overflow
int result = a * b; // Overflow!

// GOOD: Use modular arithmetic
int result = ((long long)a * b) % mod;
```

### Problem: Not Handling Coprimality

```c
// BAD: Assume inverse exists
int inv = pow(a, mod-2) % mod; // May not exist!

// GOOD: Check gcd
if (gcd(a, mod) == 1) {
    // Inverse exists
}
```
