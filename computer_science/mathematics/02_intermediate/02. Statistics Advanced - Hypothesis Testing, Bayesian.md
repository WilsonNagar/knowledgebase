## Hypothesis Testing

### What is Hypothesis Testing?

**Hypothesis Testing**: Test claims about population

**Process**: 
```
1. State null hypothesis H₀
2. State alternative hypothesis H₁
3. Choose significance level α
4. Collect data
5. Compute test statistic
6. Make decision
```

**Use**: Scientific testing, A/B testing

### Hypothesis Types

**Null Hypothesis (H₀)**: 
```
Default assumption
Usually: no effect, no difference
```

**Alternative Hypothesis (H₁)**: 
```
What we want to prove
Usually: effect exists, difference exists
```

**Decision**: Reject H₀ or fail to reject H₀

## Type I and Type II Errors

### Error Types

**Type I Error (α)**: 
```
Reject H₀ when H₀ is true
False positive
```

**Type II Error (β)**: 
```
Fail to reject H₀ when H₁ is true
False negative
```

**Power**: 
```
Power = 1 - β
Probability of rejecting H₀ when H₁ is true
```

### Error Trade-offs

**Trade-off**: 
```
Decrease α → Increase β
Increase α → Decrease β
```

**Balance**: Choose appropriate α

## P-Values

### What is a P-Value?

**P-Value**: Probability of observing data or more extreme

**Interpretation**: 
```
P-value < α → Reject H₀
P-value ≥ α → Fail to reject H₀
```

**Common α**: 0.05, 0.01

**Use**: Measure evidence against H₀

### P-Value Example

**Example**: 
```
H₀: μ = 100
Test statistic: z = 2.5
P-value: P(Z > 2.5) = 0.0062
α = 0.05
Decision: Reject H₀ (0.0062 < 0.05)
```

**Meaning**: Strong evidence against H₀

## Confidence Intervals

### What are Confidence Intervals?

**Confidence Interval**: Range containing parameter

**Interpretation**: 
```
95% CI: 95% confidence parameter in interval
```

**Formula**: 
```
CI = estimate ± margin of error
```

**Use**: Estimate parameters

### Confidence Interval Example

**Example**: 
```
Sample mean: x̄ = 50
Standard error: SE = 2
95% CI: 50 ± 1.96 × 2 = [46.08, 53.92]
```

**Meaning**: 95% confident μ in [46.08, 53.92]

## Bayesian Inference

### What is Bayesian Inference?

**Bayesian Inference**: Update beliefs with data

**Method**: 
```
1. Start with prior belief
2. Collect data
3. Update to posterior belief
```

**Use**: Learning from data

### Bayesian vs Frequentist

**Frequentist**: 
```
Parameters fixed
Probability = long-run frequency
```

**Bayesian**: 
```
Parameters random
Probability = degree of belief
```

**Difference**: Interpretation of probability

## Bayes' Theorem

### What is Bayes' Theorem?

**Bayes' Theorem**: 
```
P(A|B) = P(B|A) × P(A) / P(B)
```

**In Inference**: 
```
P(θ|data) = P(data|θ) × P(θ) / P(data)
```

**Where**: 
- **P(θ|data)**: Posterior
- **P(data|θ)**: Likelihood
- **P(θ)**: Prior
- **P(data)**: Evidence

**Use**: Update beliefs

### Bayes' Theorem Example

**Example**: 
```
Prior: P(disease) = 0.01
Test: P(positive|disease) = 0.99
      P(positive|no disease) = 0.05
Posterior: P(disease|positive) = ?
```

**Calculation**: 
```
P(disease|positive) = 
  0.99 × 0.01 / (0.99×0.01 + 0.05×0.99)
  = 0.167
```

## Prior and Posterior

### Prior Distribution

**Prior**: Belief before data

**Types**: 
- **Informative**: Based on knowledge
- **Non-informative**: Weak beliefs
- **Conjugate**: Matches likelihood

**Use**: Express initial beliefs

### Posterior Distribution

**Posterior**: Belief after data

**Computation**: 
```
Posterior ∝ Likelihood × Prior
```

**Use**: Updated beliefs, predictions

## Common Pitfalls

### Problem: Misinterpreting P-Values

```c
// BAD: P-value = probability H₀ true
// Incorrect interpretation

// GOOD: P-value = probability of data given H₀
// Correct interpretation
```

   - **A)** Data collection

   - **C)** Data visualization

2. What is a Type I error?
   - **A)** False negative
   - **B)** False positive - rejecting H₀ when H₀ is true
   - **C)** Correct rejection
   - **D)** Correct acceptance

3. What is Bayesian inference?
   - **A)** Fixed parameters
   - **B)** Updating beliefs about parameters using data and prior knowledge
   - **C)** Frequentist approach
   - **D)** Hypothesis testing

**Answers:**
1. **B** - Hypothesis testing uses sample data to test claims (hypotheses) about population parameters with controlled error rates

3. **B** - Bayesian inference updates prior beliefs about parameters to posterior beliefs using observed data via Bayes' theorem
