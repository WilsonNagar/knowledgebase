## Quiz

1. What is the shared responsibility model?
   - **A)** Customer responsible for everything
   - **B)** Division of security responsibilities between cloud provider and customer
   - **C)** Provider responsible for everything
   - **D)** No responsibility

2. What is cloud IAM?
   - **A)** Infrastructure management
   - **B)** Identity and Access Management for cloud resources
   - **C)** Application management
   - **D)** Data management

3. What is encryption at rest?
   - **A)** Encrypting data in transit
   - **B)** Encrypting stored data
   - **C)** Encrypting network traffic
   - **D)** Encrypting applications

**Answers:**
1. **B** - The shared responsibility model divides security responsibilities: cloud provider secures infrastructure, customer secures applications and data
2. **B** - Cloud IAM (Identity and Access Management) controls who can access cloud resources and what they can do, using users, roles, and policies
3. **B** - Encryption at rest encrypts data stored in cloud storage (databases, object storage) to protect it from unauthorized access
