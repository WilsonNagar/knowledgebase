# New Relic APM

## Overview

New Relic is an application performance monitoring (APM) platform. This guide covers application instrumentation, performance monitoring, error tracking, distributed tracing, custom metrics, and building comprehensive observability with New Relic.

## Deep Explanation

### Key Features

**Application Performance**: Response times, throughput
**Error Tracking**: Exception monitoring
**Distributed Tracing**: Request flow across services
**Custom Metrics**: Business metrics
**Alerts**: Performance and error alerts

### Instrumentation

**Automatic**: Framework support
**Manual**: Custom instrumentation
**APM Agents**: Language-specific agents

### Key Metrics

**Apdex**: Application performance index
**Throughput**: Requests per minute
**Response Time**: Average response time
**Error Rate**: Percentage of errors

## Real Code Examples

### Example: Custom Metrics

```python
import newrelic.agent

@newrelic.agent.function_trace()
def process_order(order_id):
    newrelic.agent.record_custom_metric("Orders/Processed", 1)
    # Process order
```
