# Grafana Visualization

## Overview

Grafana is an open-source analytics and visualization platform. This guide covers creating dashboards, data sources configuration, panel types, alerting, templating, and best practices for building effective monitoring dashboards.

## Deep Explanation

### Data Sources

**Prometheus**: Time-series metrics
**InfluxDB**: Time-series database
**MySQL/PostgreSQL**: Relational databases
**CloudWatch**: AWS metrics
**Loki**: Log aggregation

### Dashboard Creation

**Panels**:
- Time series
- Stat
- Table
- Gauge
- Bar chart
- Heatmap

**Queries**:
```promql
rate(http_requests_total[5m])
sum by (instance) (cpu_usage_percent)
```

### Alerting

**Alert Rules**:
- Threshold-based
- PromQL expressions
- Notification channels
- Alert states

## Real Code Examples

### Example: API Dashboard

**Panel Configuration**:
```json
{
  "targets": [
    {
      "expr": "rate(http_requests_total[5m])",
      "legendFormat": "{{method}} {{status}}"
    }
  ],
  "yaxes": [
    {"format": "reqps"},
    {"format": "short"}
  ]
}
```
